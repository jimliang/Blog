关于本服务器的改造计划

发表于 2016-06-26 | 分类于 脑洞 | | 阅读次数:

序

鉴于本人是一位极度懒惰兼患有严重拖延症患者，虽然脑海中一堆乱七八糟的想法，咸的辣的酸的各不相同，但事实上付出执行的没有多少。因此这个计划仅供参考，最终解释权归本人所有。

服务器概况

本服的初衷是用来翻墙的（现在还是，将来考虑弄做ss帐号管理器来分发流量），原先使用lantern但近期稳定性不好，于是自己搭了一个ss服来翻墙。又想不能因此浪费资源，便用来折腾一些小玩意。

Server Info:

LOCATION：US, California
RAM：256 MB
SWAP：128 MB
OS：Centos 6 x86

瞧！多么小巧的服务器，跑Java肯定是不行了，不过跑跑node、py是肯定没问题。

正在跑的东东

  • 小博客，发牢骚的地方 http://notes.jimliang.com/ (已切换至 Github Pages)
  • api-server，用来作数据接口 http://api.jimliang.com/
  • static-server，静态资源服务端 http://static.jimliang.com/
  • mongodb，数据存储
